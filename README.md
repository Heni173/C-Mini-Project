# 🔢✨ NUMBER SYSTEM CONVERTER ✨🔢

## 🧠 PROJECT OVERVIEW
This is a console-based Number System Converter developed in C++, which converts a decimal number into **Binary, Octal, and Hexadecimal** formats.

It demonstrates how powerful **Object-Oriented Programming (OOP)** concepts can be used to design a clean, modular, and efficient system.

---

## 🎯 OBJECTIVE
🔢 Build a real-world number conversion utility  
🧠 Apply OOP concepts in a practical project  
⚙️ Implement efficient base conversion logic  
📊 Improve problem-solving and programming skills  

---

## ⚙️ FEATURES
🔁 Convert Decimal → Binary  
🔁 Convert Decimal → Octal  
🔁 Convert Decimal → Hexadecimal  
🧩 Clean object-oriented structure  
🛡️ Input validation for error handling  
📋 Menu-driven user interface  
🔄 Continuous execution until user exits  

---

## 🧠 CORE LOGIC
The program uses a simple and efficient method:

➗ Repeatedly divide the number by the target base  
🔢 Store the remainders  
🔄 Reverse the result to get final output  

👉 This method ensures accurate and optimized conversion.

---

## 🕹️ HOW TO USE
Enter a decimal number  
Select conversion type:

1️⃣ Binary (Base 2)  
2️⃣ Octal (Base 8)  
3️⃣ Hexadecimal (Base 16)  

👉 The program displays the converted result instantly.

## 🧱 OOP CONCEPTS USED

📌 **Abstraction**  
Implemented using abstract base class `Converter`  

📌 **Inheritance**  
Derived classes: `BinaryConverter`, `OctalConverter`, `HexConverter`  

📌 **Polymorphism**  
Virtual function `convert()` enables dynamic behavior  

📌 **Function Overloading**  
`printHeader()` used in multiple forms  

📌 **Dynamic Binding**  
Handled via base class pointer (`Converter* ptr`)
